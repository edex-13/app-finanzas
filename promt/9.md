Ya tengo funcionando la aplicación financiera principal.

Ahora necesito convertirla correctamente en PWA instalable, especialmente pensando en iPhone.

Objetivo:
Configurar la app como Progressive Web App para que pueda instalarse desde Safari en iPhone usando “Agregar a pantalla de inicio”.

Tareas:

1. Instalar y configurar vite-plugin-pwa.

2. Crear manifest:
   - name
   - short_name
   - description
   - start_url
   - display standalone
   - background_color
   - theme_color
   - icons
   - orientación recomendada
   - categorías

3. Configurar iconos:
   - 192x192
   - 512x512
   - apple-touch-icon
   - favicon
   - maskable icon si aplica

4. Configurar soporte iOS:
   - apple-mobile-web-app-capable
   - apple-mobile-web-app-title
   - apple-mobile-web-app-status-bar-style
   - apple-touch-icon
   - meta viewport correcto

5. Configurar service worker:
   - cache básico de assets.
   - fallback offline.
   - no cachear datos sensibles financieros de manera insegura.
   - evitar guardar información financiera privada en caché persistente.
   - cachear solo recursos estáticos.

6. Crear pantalla offline:
   - Mensaje claro si no hay conexión.
   - Indicar que algunos datos requieren conexión.
   - Mantener navegación básica.

7. Crear componente InstallPrompt:
   - En iPhone mostrar instrucciones:
     1. Abrir en Safari.
     2. Tocar compartir.
     3. Tocar “Agregar a pantalla de inicio”.
   - En Android usar beforeinstallprompt si está disponible.
   - En desktop mostrar mensaje discreto.

8. Revisar que la app funcione bien en modo standalone.

9. Agregar checklist de pruebas:
   - Safari iPhone.
   - Chrome Android.
   - Desktop.
   - Recarga offline.
   - Login.
   - Dashboard.
   - Rutas protegidas.

10. Entregable:
   - Configuración Vite.
   - manifest.
   - service worker.
   - componente de instalación.
   - cambios en index.html.
   - recomendaciones de seguridad para no cachear datos financieros sensibles.
